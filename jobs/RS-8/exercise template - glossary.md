muscle_group - i.e. chest, back, lumbar back etc
variant - i.e. barbell, dumbbell, close-grip, wide-grip etc
next_exercise_milestone - my monthly or quarterly goal. What I'm working towards achieving. Can be associated with the muscle group or the variant.
exercise_name - i.e. bench press, chest press, etc.
today_main_set - main set of the day, i.e. 3x10x100 lb
PR = personal record
1_rm = one rep max. Typically calculated from a whole set.
set_volume_pr = pr for most volume for a single set for a given exercise variant. i.e. for a single main set.
exercise_volume_pr = pr for most volume for the whole exercise, including warmup sets, main sets, and cool down sets. 
set_type = i.e. warmup, main set, or cooldown
type_id = i.e. which set of a particular type. Typcially 1,2,3,4,5...
set_details = i.e. 10 x 100 lb
barbell_weight_load = for a barbell, typically gyms will have weight in the following increments: 2.5, 5, 10, 25, 45 lb. The barbell weighs 45 lb. This shows how much weight each side of the barbell should be loaded with, in order to achieve the proper total loading for the set.
last_set_details =  The last time I did this exercise. what did I do for this set type and ID? what date was it?
intensity = denotes the rep range: strength (typically 3-6 reps), hypertrophy (typically 7-11 reps), endurance (12-15 reps)
